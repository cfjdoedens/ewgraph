# Probability density function for ew_graph

Approximate the density of an ew_graph anywhere on the line \[0, 1\]. In
other words return the (approximated) chance density of the ew_graph g
for any value of p on \[0, 1\].

## Usage

``` r
ew_d(g, p)
```

## Arguments

- g:

  An ew_graph.

- p:

  A real value from \[0, 1\].

## Value

The (approximated) chance density for p on g.
